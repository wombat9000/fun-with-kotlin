
#### Setup

- Clone the Repo
```
git clone https://github.com/wombat9000/fun-with-kotlin.git
```
- Initialise the Project
```
./gradlew idea
```
- Open the .ipr file with IntelliJ.
- Install/Update Kotlin Plugin
